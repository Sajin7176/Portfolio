# Personal Portfolio — Sajin S

Simple, responsive personal portfolio scaffold.

Files:
- `index.html` — main page
- `styles.css` — styling
- `script.js` — small interactions

How to view:
1. Open `index.html` in a browser (double-click or use Live Server).

Customize:
- Replace placeholder links and projects with your real content.
- Add images to an `assets/` folder if needed.

Design notes: clean, modern, professional colors (white, dark blue, light grey), rounded cards, subtle shadows, smooth scroll.
